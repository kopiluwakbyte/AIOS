Konfigurasi API Diperlukan
Aplikasi ini berjalan secara statis di GitHub Pages. Hubungkan ke database Google Sheets dengan memasukkan URL Web App Google Apps Script Anda.
1
Buka file spreadsheet Google Sheets Anda, pilih menu Ekstensi > Apps Script.
2
Salin kode dari code.gs ke editor Apps Script, lalu klik simpan.
3
Klik tombol Terapkan (Deploy) > Terapkan Baru (New deployment). Pilih jenis Aplikasi Web (Web App). Setel akses "Siapa yang dapat mengakses" menjadi Siapa saja (Anyone).
4
Salin URL Aplikasi Web yang dihasilkan dan tempelkan pada kolom di bawah ini.
